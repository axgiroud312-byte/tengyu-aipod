# Temu 采集

Temu 采集的目标是把搜索页、列表页或商品详情页里的图片先放进图池，再由你选择下载到 `01-采集工作区`。

![Temu 图池采集界面](/tutorial/collection-temu.png)

## 前置条件

1. 比特浏览器已经启动。
2. 浏览器环境里已经登录 Temu。
3. 设置页里保存了比特浏览器地址。
4. 左侧进入「采集」，平台选择 `Temu`。

## 打开 Temu 页面

你可以从两种页面开始：

| 页面 | 适合场景 |
|---|---|
| 搜索页 / 列表页 | 想快速扫一批候选图。 |
| 商品详情页 | 想采集某个商品的主图轮播。 |

在采集页填写搜索关键词后，可以点击打开搜索页；也可以直接在比特浏览器里手动打开目标 Temu 页面。

## 扫描图池

1. 选择「浏览器环境」。
2. 确认 Temu 页面已经在比特浏览器里打开。
3. 点击「扫描图池」。
4. 等待图池列表出现图片。
5. 勾选要下载的图片。
6. 点击「下载选中」或「下载全部」。

扫描不会立刻写文件，只有下载时才会落盘。

## 保存目录

下载成功后，图片会进入：

```text
01-采集工作区/temu-时间/
```

商品详情页主图会尽量按商品分组保存：

```text
01-采集工作区/temu-时间/商品页/temu-g-商品ID/
```

搜索页和列表页的散图会直接保存在当次任务目录里。

## 常见问题

| 现象 | 处理方式 |
|---|---|
| 扫描不到图片 | 确认当前比特浏览器页面属于 Temu，并且页面已经加载完成。 |
| 下载失败 | 打开采集日志，看是网络失败、图片地址失效，还是平台限制。 |
| 当前页不对 | 先在比特浏览器切到目标 Temu 标签页，再回到腾域点击扫描。 |
| 遇到验证码 | 手动在比特浏览器里处理。腾域不会绕过验证码或登录风控。 |
