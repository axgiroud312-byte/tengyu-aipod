# PS 套版

PS 套版会把 `02-印花工作区` 里的印花批量替换到 PSD/PSB 模板的智能对象里，并输出到 `04-上架工作区`。

![PS 套版界面](/tutorial/photoshop-batch-mockup.png)

## 前置条件

1. 使用 Windows 电脑。
2. 已安装 Photoshop。
3. Photoshop 可以正常启动。
4. PSD/PSB 模板里有可替换的智能对象。
5. `02-印花工作区` 里已有可套版图片。

macOS 上可以查看页面结构，但不能实际执行 Photoshop COM 套版。

## 选择输入图片

1. 进入「PS 套版」。
2. 在「印花文件夹」里选择输入目录。
3. 默认可以使用 `02-印花工作区`。
4. 点击检索后，确认套版候选清单里有印花。

## 扫描模板

1. 在「PSD / PSB 模板」里选择一个或多个模板。
2. 点击「扫描模板」。
3. 等待模板预览出现。
4. 确认模板数、印花数、裁切数没有异常。

扫描会识别智能对象和裁切区域。第一次扫描较慢，后续命中缓存会更快。

## 输出参数

| 参数 | 建议 |
|---|---|
| 替换范围 | 普通模板优先用「顶层智能对象」。 |
| 裁切模式 | 优先用「自动裁切」。模板有明确参考线时可用「参考辅助线」。 |
| 输出目录 | 默认保存到 `04-上架工作区/套版-时间`。 |

输出目录可以直接作为后续标题生成和上架读取的批次目录。

## 开始套版

1. 确认 Photoshop 状态正常。
2. 确认印花数量和模板数量正确。
3. 点击「开始套版」。
4. 运行中可以打开日志查看当前模板和当前印花。
5. 完成后点击「打开输出目录」检查结果。

## 常见问题

| 现象 | 处理方式 |
|---|---|
| 提示仅 Windows 可用 | 切到 Windows 电脑执行。 |
| Photoshop 状态异常 | 先手动启动 Photoshop，再刷新状态。 |
| 模板扫描失败 | 检查 PSD/PSB 是否损坏，是否包含智能对象。 |
| 输出数量不对 | 看日志里的模板、印花、裁切数，再检查输出目录。 |
