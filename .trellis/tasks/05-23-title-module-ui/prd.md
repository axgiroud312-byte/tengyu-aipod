# Task: 标题模块前端 UI（切片 2 - 标题生成模块）

## 目标

标题模块的 UI 表单 + 进度面板 + 失败列表。

## 输入

参考文档（按重要性排序）：
- `docs/spec/06-title.md §5`

## 验收标准

- [ ] 5+1 输入：批次路径 / 平台 / 语言 / 模型 / 取第几张 / 标题额外要求
- [ ] 已有标题策略 radio（跳过 / 重新生成）
- [ ] 失败重试次数 + 并发数
- [ ] 图像预处理开关（透明底强制 + 压缩可选）
- [ ] 预估费用实时显示（按图数 × 模型单价）
- [ ] [开始生成标题] 按钮 → 调 IPC `title:run` → 进度面板
- [ ] 进度：处理中 / 跳过 / 失败重试中 / 完成
- [ ] 完成后：成功列表 / 失败列表（带[重试失败]）
- [ ] [打开 xlsx] / [打开批次目录] 按钮

## 不做

- 不实现编辑标题 UI（v1 不要）
- 不实现 GPT-4 备用切换

## 实施提示

平台/语言/模型下拉都来自云端 skill cache（不写死）。
