# 生图常用三项

初版教程只覆盖最常用的 3 条路径：文生图、图生图、提取。抠图和提取后抠图后续单独补教程。

![生图常用路径界面](/tutorial/generation-common-paths.png)

## 路径怎么选

| 入口 | 用途 | 输出目录 |
|---|---|---|
| 文生图 | 从文字提示词直接生成新印花。 | `02-印花工作区/文生图/任务名/` |
| 图生图 | 用参考图控制风格或构图，再生成新印花。 | `02-印花工作区/图生图/任务名/` |
| 提取 | 从采集图里提取印花元素。 | `02-印花工作区/提取/任务名/` |

## 文生图

1. 进入「生图」。
2. 选择「文生图」。
3. 选择「智能生成提示词」或「自己写提示词」。
4. 智能生成时，填写需求和提示词数量，然后点击「生成提示词」。
5. 在「提示词审稿」里保留要生成的提示词。
6. 选择 Grsai 或 ComfyUI 工作流路径。
7. 填写任务名，或使用默认任务名。
8. 点击「开始生图」。

如果只是快速出图，优先用 Grsai。需要使用本地或晨羽工作流时，再切到 ComfyUI 工作流。

## 图生图

1. 进入「图生图」。
2. 选择参考图或图片文件夹。
3. Grsai 路径里选择参考方式：构图、风格、构图加风格，或自己写提示词。
4. ComfyUI 路径里选择图生图工作流和运行云机。
5. 填写任务名。
6. 点击「开始图生图」。

图生图适合保留某张图的画面方向，但不要直接复制原图主体。

## 提取

1. 进入「提取」。
2. 选择采集图所在文件夹，通常来自 `01-采集工作区`。
3. 选择提取 Skill。
4. Grsai 路径里选择生图模型；ComfyUI 路径里选择提取工作流和运行云机。
5. 填写任务名。
6. 点击「开始提取」。

提取后的图片才算印花，可以继续进入 PS 套版。

## 失败时先看这里

| 现象 | 处理方式 |
|---|---|
| 生成提示词失败 | 检查百炼 API Key、模型、日志里的原始返回。 |
| Grsai 不出图 | 检查 Grsai API Key、节点、余额和生图日志。 |
| ComfyUI 不能开始 | 确认有运行中的云机，并且选了对应工作流。 |
| 输出找不到 | 在结果卡片里看保存路径，或到对应能力目录下找任务名文件夹。 |
